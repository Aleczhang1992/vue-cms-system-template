# 作业盒子磐石平台（盒图）
***说明：***
磐石系统是公司大数据分析的基础平台
## Environment

`Node >= 6`

## Start

 - Clone or download this repository
 - Enter your local directory, and install dependencies:


``` bash
yarn
```

## Develop

``` bash
# serve with hot reload at localhost:8010
npm run dev
```

## Build

``` bash
# build for production with minification
npm run build   
npm run build:all
```
